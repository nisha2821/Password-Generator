# Password Generator

## Description

A Python project that generates strong random passwords.

## Features

* Generates random passwords
* Includes uppercase letters
* Includes lowercase letters
* Includes numbers
* Includes special characters
* User can choose password length

## Technologies Used

* Python 3
* Random Module

## How It Works

1. User enters the password length
2. Program ensures at least:
   - One uppercase letter
   - One lowercase letter
   - One number
   - One special character
3. Remaining characters are generated randomly
4. Password is shuffled
5. Strong password is displayed

## Example Output

Enter password length: 10

Generated Password: K7@aP2#mQ9
